## 💡 面试建议

### 诚实但自信

**项目状态：**
- ✅ 已完成：产品调研、方案设计、代码开发、Mock数据测试
- 🔄 进行中：联系种子用户，准备上线前验证
- ⏳ 待启动：真实数据测试、小范围试运行

**话术模板：**
> "这是我独立设计并开发的AI产品项目。我先访谈了5位目标用户发现痛点，然后设计了三大AI功能并完成开发，用Mock数据测试验证效果。目前正在联系救助站准备真实数据验证。这个项目展示了我的产品设计、技术实现和数据分析能力。"

### 准备3个故事

1. **用户洞察故事**：小李的退养案例→时间投入维度设计
2. **产品设计故事**：问卷从10轮到6-8轮的迭代→平衡信息量和完成率
3. **技术选型故事**：LongCat vs GPT-4→成本、稳定性、中文效果的权衡

### 准备1个失败案例

> "我最初考虑直接询问用户收入，但访谈中发现隐私敏感度极高。改为询问职业类型间接推断，既获取了必要信息又降低了用户防备。这个教训让我明白产品设计要考虑用户心理。"
